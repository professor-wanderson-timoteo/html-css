Título Princial - h1

O "h1" é o título principal e mais importante da página.

Subtítulo - h2

O "h2" é um subtítulo, usado para dividir seções dentro de página

Seção menor - h3

O "h3" é um nível abaixo do "h2", geralmente usado para subseções dentro de um tópico.

Seção ainda menor - h4

O "h4" é menos importante, geralmente usado para destacar partes dentro de um "h3".

Detalhes - h5

O "h5" pode ser usado para títulos de menos relevância, como informações complementares

Último nível - h6

O "h6" é o menor título disponível e geralmente usado para legendas e notas.
